Waarp Password GUI and Console Management

WaarpPassword is an helper to construct Password in compatible way for Waarp products.

Dependency

This project used Swing for GUI and Waarp Common Crypto package for cryptographic support.

Specificities

Two ways are available to use it: through the graphical interface (WaarpUiPassword) or through command line (WaarpPassword).

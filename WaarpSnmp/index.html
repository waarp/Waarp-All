Waarp SNMP

The Waarp SNMP support is made to provide SNMP agent support to softwares in Waarp project. It is based on SNMP4J library.

Dependency

It is based mainly on the SNMP4J library (at http://www.snmp4j.org/.

Overview

It is possible to define by a new class a new MIB.

It is possible to have SNMP notifications as trap or inform form.

It is compatible with SNMPV2c and SNMPV3 (supposely also with SNMPV1).

Network
#######

 * `server.xml`: Optionel

.. option:: serverport

   *Integer* (6666)

   Port utilisé pour les connections non chiffrées

.. option:: serversslport

   *Integer* (6667)

   Port utilisé pour les connections chiffrées

.. option:: serverhttpport

   *Integer* (8066)

   Port utilisé pour l'interface HTTP de monitoring

.. option:: serverhttpsport

   *Integer* (8067)

   Port utilisé pour l'interface HTTPS d'dministration

.. option:: REST

   *Integer* (8067)

   Port utilisé pour l'accès REST en HTTP(S)

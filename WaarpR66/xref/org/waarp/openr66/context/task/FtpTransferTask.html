35  /**
36   * Ftp Transfer task: synchronous<br>
37   * <p>
38   * Result of arguments will be as FTP command.<br>
39   * Format is the following:<br>
40   * "-file filepath <br>
41   * -to requestedHost <br>
42   * -port port <br>
43   * -user user <br>
44   * -pwd pwd <br>
45   * [-account account] <br>
46   * [-mode active/passive] <br>
47   * [-ssl no/implicit/explicit]<br>
48   * [-cwd remotepath] <br>
49   * [-digest (crc,md5,sha1)] <br>
50   * [-pre extraCommand1 with ',' as separator of arguments] <br>
51   * -command command from (get,put,append) <br>
52   * [-post extraCommand2 with ',' as separator of arguments]" <br>
53   * <br>
54   * <br>
55   * The order of commands will be:<br>
56   * 1) connection to requestHost on port (if ssl native => using native ssl
57   * link)<br>
58   * 2) User user<br>
59   * 3) PASS pwd<br>
60   * 4) if account => ACCT account<br>
61   * 5) if -ssl & auth => AUTH, PBSZ 0, PROT P <br>
62   * 6) if passive => PASV<br>
63   * 7) CWD remotepath; if error => MKD remotepath then CWD remotepath (and
64   * ignoring any error)<br>
65   * 8) if pre => extraCommand1 with ',' replaced by ' ' (note: do not use
66   * standard commands from FTP like
67   * ACCT,PASS,REIN,USER,APPE,STOR,STOU,RETR,RMD,RNFR,RNTO,ABOR,CWD,CDUP,MODE,PASV,PORT,STRU,TYPE,MDTM,MLSD,MLST,SIZE,AUTH)
68   * <br>
69   * 9) BINARY (binary format)<br>
70   * 10) if get => RETR filepath.basename ; if put => STOR filepath ; if append
71   * =>
72   * APPE filepath.basename<br>
73   * 11) if digest & get/put/append & remote site compatible with XCRC,XMD5,XSHA1
74   * => FEAT (parsing if found
75   * corresponding XCRC,XMD5,XSHA1) ; then XCRC/XMD5/XSHA1 filepath.basename ;
76   * then locally comparing this
77   * XCRC/XMD5/XSHA1 with the local file<br>
78   * 12) if post => extraCommand2 with ',' replaced by ' ' (note: do not use
79   * standard commands from FTP like
80   * ACCT,PASS,REIN,USER,APPE,STOR,STOU,RETR,RMD,RNFR,RNTO,ABOR,CWD,CDUP,MODE,PASV,PORT,STRU,TYPE,MDTM,MLSD,MLST,SIZE,AUTH)<br>
81   * 13) QUIT<br>
82   */

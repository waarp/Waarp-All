Waarp XmlEditor

The Waarp XmlEditor project proposes an easy way to configure the Waarp projects XML configuration files.

Dependency

It is based on the XmlEditor from Felix Golubov.

See global Waarp project at http://waarp.github.com/Waarp/ and particular XmlEditor project at http://github.com/WaarpXmlEditor/
